# Blix Website Clone

![HTML5](https://img.shields.io/badge/html5-%23E34F26.svg?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/css3-%231572B6.svg?style=for-the-badge&logo=css3&logoColor=white)

A pixel-perfect, responsive front-end clone of the Blix website built using pure HTML and CSS. This project was created for educational purposes to practice modern web layout techniques and responsive design.

## 🎯 Features

* **Responsive Design:** Fully fluid layout that adapts to mobile, tablet, and desktop screens using CSS media queries.
* **Modern Layouts:** Utilizes CSS Flexbox and CSS Grid for complex component positioning and alignment.
* **Semantic HTML:** Structured with accessibility and SEO best practices in mind.
* **Custom Styling:** Replicates the brand's typography, color palette, and hover animations without relying on UI frameworks like Bootstrap or Tailwind.
* **Zero Dependencies:** A lightweight project built strictly with vanilla web technologies.

## 🚀 Technologies Used

* **HTML5**
* **CSS3** (Flexbox, Grid, Variables, Animations)
